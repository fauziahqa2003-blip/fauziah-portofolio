# Personal Portfolio Website

A modern and responsive personal portfolio website showcasing my projects, professional experiences, certifications, technical skills, and academic journey.

## Features

- Responsive Design
- Interactive Navigation
- Animated Sections
- Project Showcase
- Certification Gallery
- Journey Timeline
- Skills Visualization
- Contact Information
- Modern Dark Theme

- ## Built With

- HTML5
- CSS3
- JavaScript
- AOS Animation
- Font Awesome
- Google Fonts
